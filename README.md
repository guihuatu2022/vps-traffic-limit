# 🛡️ VPS 流量监控自动锁定系统

> **超限自动锁网卡，SSH 永不断连，次月自动解锁**

---

## 这是什么？

如果你的 VPS 流量超了，云厂商会扣钱（GCP 的 200GB 免费额度用完就开始计费）。

这个工具能帮你：

| 功能 | 说明 |
|------|------|
| 🔍 **自动监控** | 每 3 分钟检查一次流量 |
| 🔒 **自动锁定** | 流量超限 → 自动阻断所有出站流量（SSH 不断） |
| 🔓 **自动解锁** | 次月 1 号自动恢复网络 |
| 📢 **自动通知** | 超限/预警时发 Telegram / 微信 / 邮件提醒 |
| 🧪 **测试模式** | 30 秒模拟锁定，验证 SSH 不会断 |

---

## 一键安装

**适用系统：** Debian 10+ / Ubuntu 20.04+（amd64 / arm64 都支持）

在 VPS 上执行：

```bash
git clone https://github.com/guihuatu2022/vps-traffic-limit.git
cd vps-traffic-limit/src
sudo bash install.sh
```

安装过程会：
1. ✅ 自动检测你的网卡和 SSH 端口
2. ✅ 自动安装所需依赖（vnstat、python3、curl）
3. ✅ 一步一步引导你配置限额和通知
4. ✅ 部署完成后立即开始监控

> 不想交互？一行搞定全部默认配置：
> ```bash
> sudo bash install.sh --yes
> ```

## 核心逻辑说明

### 锁定后什么能通什么不能通？

**严格模式（默认）：**

```
✅ SSH 端口  → 你能连上 VPS
✅ DNS       → 域名能解析
✅ DHCP      → IP 地址不会掉
❌ 其他所有   → 全拦
```

**自定义模式：** 你额外指定放行端口（如 80、443 让网站继续跑）。

### 怎么保证重启不会丢规则？

系统每次启动时会自动检查流量：
- 如果还超限 → **重新锁上**
- 如果已到次月 → **自动解锁**

### 通知怎么配？

安装时选一个：

| 通知方式 | 难度 | 说明 |
|----------|------|------|
| Telegram | ⭐ 最简单 | 创建 Bot，填 Token 即可 |
| 企业微信 | ⭐⭐ | 创建群机器人，填 Webhook URL |
| 邮件 | ⭐⭐⭐ | 需要 SMTP 账号 |
| 不通知 | — | 只写日志，不推送 |

## 常见问题

**Q：VPS 重启后锁还在吗？**

重启后规则可能会丢失（看是否装了 iptables-persistent），但系统自愈服务会在开机后**立刻重新检查**，如果流量仍超限就重新锁上。

**Q：SSH 端口不是 22 怎么办？**

安装时脚本会自动检测你的 SSH 端口（`sshd -T`），检测不到可以用 `--ssh-port` 手动指定。

**Q：GCP / AWS / 阿里云计费方式不同？**

安装时可以选择计费方向：
- **上行**（egress）— GCP、AWS、Azure、阿里云（默认）
- **双向**（total）— Hetzner、BuyVM
- **下行**（ingress）— 极少

**Q：怎么确认锁真的有效？**

```bash
llcx lock    # 锁上
curl https://www.baidu.com   # 应该超时
ssh 你的VPS   # 应该还能连
llcx unlock  # 解锁
```

或者用测试模式自动验证：

```bash
llcx         # 进菜单 → 3.锁定管理 → 3.测试锁定30秒
```
