# VPS 流量监控 — llcx 交互菜单设计方案

## 设计理念

- **入口统一**：不论查流量、查历史、配置、锁定、日志，都从 `llcx` 进
- **菜单驱动**：新手可以一步步选，不用记命令
- **快速通道**：高级用户加参数跳过菜单，一步直达

```
llcx          → 打开交互菜单（新手友好）
llcx -c       → 直接显示当前流量状态（一步直达，同旧版 llcx）
llcx -r       → 直接显示历史月度汇总（一步直达，同旧版 ydcx）
llcx lock     → 直接锁定（免菜单）
llcx unlock   → 直接解锁（免菜单）
```

## 命令速查表

| 命令 | 效果 |
|------|------|
| `llcx` | 打开交互菜单 |
| `llcx -c` 或 `llcx --check` | 直接显示本月流量（免菜单） |
| `llcx -r` 或 `llcx --report` | 直接显示历史汇总（免菜单） |
| `llcx lock` | 直接锁定 |
| `llcx unlock` | 直接解锁 |
| `llcx config` | 直接进入配置管理菜单 |
| `llcx logs` | 直接进入日志菜单 |
| `llcx test-notify` | 直接发送测试通知 |
